# BelAge 📱

An Android application designed to combat isolation among elderly people.

## Features
- Bottom navigation (Home, Famille, Jeux)
- Medication reminders with Room Database
- Warm and accessible UI designed for seniors
- Golden color palette for visual comfort

## Tech Stack
- Java / Android Studio
- Room Database
- Fragment Navigation
- Material Design
- XML Layouts
